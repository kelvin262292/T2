Chào mừng đến với Test Website

Đây là trang chủ của website test để kiểm tra crawler.

Các tính năng:

  • Trang chủ (hiện tại)
  • Trang giới thiệu
  • Trang liên hệ
  • Trang sản phẩm
  • Trang có lỗi JavaScript
  • Link không tồn tại (404)

Website này được tạo để test crawler và kiểm tra lỗi.
